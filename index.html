Sobre Mim

Estou em transição para a área de Análise e Desenvolvimento de Sistemas, com forte habilidade em resolução de problemas e pensamento analítico. Atualmente, aprimoro meus conhecimentos em tecnologias low-code para criação de chatbots (Blip, Zenvia e Manychat), além de JavaScript, Python, HTML5, CSS e SQL para desenvolvimento, e Cypress e Selenium para testes automatizados. Meu objetivo é ingressar de forma sólida no setor e contribuir ativamente para projetos inovadores.

Minha experiência em atendimento ao cliente e suporte técnico me proporciona uma visão estratégica na solução de problemas, aliando comunicação eficaz, análise crítica e adaptabilidade às necessidades do negócio.

Projeto Front End

Este projeto contém diversos exercícios e aplicações práticas externas para o desenvolvimento front-end. São abordadas tecnologias essenciais como HTML, CSS e JavaScript, fornecendo uma base sólida para a criação de interfaces web. Além dos conceitos básicos, o repositório pode incluir técnicas mais avançadas, como manipulação do DOM, responsividade e interatividade com eventos. Esse material é útil tanto para iniciantes quanto para aqueles que desejam aprimorar seus conhecimentos.
